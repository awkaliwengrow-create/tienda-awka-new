# Awka Liwen Growshop

Sitio e-commerce estático para **Awka Liwen**, growshop natural con sede en Buenos Aires (desde 2020).

## Stack

- HTML5 semántico
- CSS3 con custom properties (design tokens)
- JavaScript vanilla (sin frameworks)
- Google Fonts: Covered By Your Grace + Crimson Pro

## Estructura

```
awka-liwen-sitio/
├── index.html          # Landing page
├── shop.html           # Catálogo con filtros y carrito
├── css/
│   └── styles.css      # Estilos compartidos
├── js/
│   ├── products.js     # Catálogo de productos (data)
│   └── shop.js         # Lógica de carrito, filtros, modal
└── img/
    ├── hero-grow.jpg
    ├── banners/
    ├── categorias/
    └── productos/
```

## Deploy

El sitio es 100% estático y puede hostearse en:

- **GitHub Pages** — gratuito, desde el mismo repo
- **Netlify** — drag & drop o conectado al repo
- **Vercel** — conectado al repo

### GitHub Pages (recomendado)

1. Push al repo
2. Settings → Pages → Branch: `main` → `/` (root)
3. URL: `https://<usuario>.github.io/awka-liwen-sitio`
